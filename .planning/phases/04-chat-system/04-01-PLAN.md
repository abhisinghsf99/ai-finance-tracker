<task type="auto">
  <name>Task 2: Create chat API route and message component</name>
  <files>fintrack-dashboard/src/app/api/chat/route.ts, fintrack-dashboard/src/components/chat/chat-message.tsx</files>
  <action>
1. Create `src/app/api/chat/route.ts`:
   - Export `maxDuration = 300` at top level (Vercel Fluid Compute)
   - Export async POST handler that:
     a. Reads `{ messages }` from request JSON (typed as `UIMessage[]` from 'ai')
     b. Creates MCP client per-request using `createMCPClient` from `@ai-sdk/mcp` with `StreamableHTTPClientTransport` from `@modelcontextprotocol/sdk/client/streamableHttp.js` pointing to `https://claudefinancetracker.xyz/mcp`
     c. Gets tools via `mcpClient.tools()`
     d. Calls `streamText` from 'ai' with:
        - model: `anthropic('claude-sonnet-4-20250514')` from `@ai-sdk/anthropic`
        - system: SYSTEM_PROMPT from `@/lib/chat-config`
        - messages: `convertToModelMessages(messages)` (use `await` -- it returns a promise)
        - tools from MCP client
        - `stopWhen: stepCountIs(3)` for multi-step tool call limit
        - `onFinish`: close MCP client via `mcpClient.close()`
     e. Returns `result.toUIMessageStreamResponse()`
   - Do NOT use Edge runtime (no `export const runtime = 'edge'`)
   - Do NOT create MCP client at module scope

2. Create `src/components/chat/chat-message.tsx`:
   - "use client" directive
   - Import `UIMessage` type from 'ai' (for typing only)
   - Import `ReactMarkdown` from 'react-markdown' and `remarkGfm` from 'remark-gfm'
   - Export `ChatMessage` component accepting `{ message: UIMessage }`
   - Determine `isUser` from `message.role === 'user'`
   - Render a flex container with `justify-end` for user, `justify-start` for assistant
   - Message bubble: `max-w-[85%] rounded-2xl px-4 py-3`
     - User: `bg-primary text-primary-foreground` (teal accent)
     - Assistant: `bg-muted`
   - Iterate `message.parts` -- for `type === 'text'`:
     - User messages: render as plain `<p>` tag
     - Assistant messages: render with `<ReactMarkdown remarkPlugins={[remarkGfm]}>` with custom component overrides for:
       - `table`: wrapped in `overflow-x-auto my-2` div, table gets `w-full text-sm border-collapse`
       - `th`: `border-b border-border px-3 py-1.5 text-left font-medium`
       - `td`: `border-b border-border/50 px-3 py-1.5`
       - `p`: `mb-2 last:mb-0`
       - `strong`: `font-semibold`
   - Use `React.memo` on the component export for streaming performance (per RESEARCH.md Pitfall 4)
   - Skip non-text parts (tool-call, tool-result) -- do not render them
  </action>
